# expense-tracker-python
Simple Expense Tracker built with Python and pandas
A simple expense tracking program built using *Python* and *pandas*.  
This program allows users to record their daily expenses, view a summary,  
and calculate total spending. All data is stored permanently in a CSV file.

---

## Features
- Add new expenses (with date, name, amount, and category)
- View all expenses neatly in a table
- Calculate total amount spent
- Data automatically saved to a CSV file

---

## Tech Used
- Python 3
- Pandas library
- CSV file handling

---

## How to Run
1. Clone or download this repository  
2. Open the folder in VS Code or your terminal  
3. Install pandas
